PC-Komponenter

CPU

CPU eller Prossesor er den delen i PC-en som all data proseseres igjennom. den fungererer som hjernen i opperasjonen.

Les mer

Grafikkort

Grafikkort eller GPU er den delen i pc-en som lager bildet på pcen og 3D-rendere spill og sånt, sånn at du får en fin opplevelse

Les mer

Core (kjerne)

En core er en fysisk prosesseringsenhet inne i CPU-en. Hver core kan utføre egne oppgaver samtidig, noe som gjør at datamaskinen kan gjøre flere ting på én gang.

Les mer

RAM

RAM (Random Access Memory) er datamaskinens midlertidige arbeidsminne. Det lagrer data og programmer som er i bruk akkurat nå, slik at CPU-en får rask tilgang til dem. Jo mer RAM en PC har, jo flere programmer og oppgaver kan kjøres samtidig uten at systemet blir tregt.

Les mer

Thread (tråd)

CPU-tråder (threads) er de virtuelle instruksjonslinjene som en prosessorkjerne bruker for å håndtere oppgaver. Mens fysiske kjerner gjør det tunge løftearbeidet.

Les mer

HDD

HDD (Hard Disk Drive) er en lagringsenhet som bruker en roterende disk til å lagre data. Den er tregere enn SSD, men gir mye lagring for lav pris, derfor brukes ofte HDD til filer og SSD til Windows.

Les mer

SSD

SSD er en moderne lagringsenhet uten bevegelige deler. Den er mye raskere enn HDD og brukes ofte til operativsystem og programmer. fordelene med SSD er at oppstartstiden på OS eller apper er mye raskere enn det det hadde vært på en HDD

Les mer

Hovedkort

Hovedkortet (motherboard) er den PC-komponenten som kobler til alle komponentene og er "mellommannen" når komponentene skal gi beskjed til hverandre.

Les mer

PCIe

PCIe er en tilkobling på hovedkortet for grafikkort og andre komponenter. Ulike versjoner og størrelser bestemmer hastigheten.

Les mer

AMD sine CPU

AMD Ryzen 3, 5, 7 og 9

Tallene i Ryzen-navnene viser ytelsesnivå, ikke eksakt antall cores.

Ryzen 3: inngangsnivå, færre cores, beregnet for enkel bruk

Ryzen 5: mellomklasse, ofte 6 cores, egnet for gaming og skole

Ryzen 7: høy ytelse, ofte 8 cores, bra for multitasking

Ryzen 9: toppnivå, mange cores (12–16), beregnet for tunge oppgaver

Les mer

Intel sine CPU

Intel i3, i5, i7 og i9

Intel bruker samme type klassesystem.

i3: lav ytelse, få cores

i5: mellomklasse, ofte brukt til gaming

i7: kraftig prosessor med flere cores og threads

i9: high-end, svært høy ytelse

Les mer

PSU

PSU (Power Supply Unit) er strømforsyningen i PC-en. Den gjør strøm fra stikkontakten om til riktig type og spenning som komponentene i PC-en trenger. PSU-en fordeler strøm til blant annet hovedkort, CPU, GPU, lagring og vifter.

Les mer

NPU

NPU (Neural Processing Unit) er en prosessor som er laget for kunstig intelligens og maskinlæring. Den brukes til oppgaver som bilde- og talegjenkjenning, og finnes ofte i nyere CPU-er.

Les mer

I/O

I/O står for Input og Output, altså hvordan data går inn og ut av datamaskinen. Eksempler er tastatur, mus, skjerm, USB-porter og nettverk.
